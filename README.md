## 介绍

🎉 彩蛋指令即刻触发


在 uTools 输入框中键入【礼花】或【撒花】，立即触发全屏电子礼花特效，为你的高光时刻增添仪式感：

💻 代码零错误通过编译

🎯 设计方案一稿过审

🚀 团队冲刺提前完成

✨ 任何值得标记的微小成就
